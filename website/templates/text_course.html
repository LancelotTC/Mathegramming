Derivatives in Calculus

Calculus is the study of functions, and one useful attribute to know about a function is how fast it changes. Recall that the slope of a function describes how fast the function changes, and the slope of a function can be found with the formula m = y 1 - y 0 x 1 - x 0 where m stands for slope, ( x 0 , y 0 ) is the starting point and ( x 1 , y 1 ) is the endpoint of the interval in question. This formula looks at the long-term behavior of a function between two discrete points, such as x 0 = 1.0 and x 1 = 2.0 . Differential calculus, on the other hand, studies the instantaneous rate of change of a function. This means that differential calculus studies the change in a function between two continuous points such as x 0 = 1.0 and x 0 = 1.000000000000000001 . For example, the velocity of an object is a measure of the instantaneous rate of change of the object's displacement, and an object's acceleration is a measure of the instantaneous rate of change of the object's velocity.

Computing Derivatives

To move away from the notion of the slope of a line which describes the discrete, long-term behavior of a function, the first step is to think about taking the limit of a function. Taking the limit of a function means to look at the behavior of a function as the function approaches a given point, and, when taking a limit, it is important to specify if the limit is being taken from - ∞ or ∞ . To take a limit from - ∞ or ∞ means to start at either the far left of the function or the far right of the function, and then to trace the function to the given point. The value that the function approaches as the point is reached is the limit of the function if the function approaches the same value from both - ∞ and ∞

This method for taking the limit of a function is not enough to describe the instantaneous change in the function, however.
